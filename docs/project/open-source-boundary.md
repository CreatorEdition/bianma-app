# bianma-app 公有仓边界说明

## 文档目的

本说明用于定义 `bianma-app` 公有仓（本仓库）的公开定位与边界，帮助贡献者和用户理解“这里应该放什么，不应该放什么”。

## 公有仓承载范围

本仓库主要承载：

- 开源核心能力的代码与文档
- 社区协作与贡献流程
- 公开协议与公开接入文档
- 迁移与兼容说明
- 用户手册与 FAQ

## 不在公有仓展开的内容

以下内容不应在本仓库作为主文档展开：

- 私有发布流程
- 私有 updater / 私有分发细节
- 内部治理文档
- 内部 roadmap
- 商业合作策略
- 私有渠道策略
- 未准备开源的产品增强细节

## 品牌与协议口径

- 公有仓文档使用官方品牌 `bianma-app`。
- 公有仓主协议口径为 `bianma://`。
- `ccswitch://` 仅用于迁移与 legacy compatibility 说明，不作为主协议宣传。
- 工具支持范围以公开文档已标注清单为准；未列入“已支持”的工具不作公开支持承诺。

## 与官方产品层的关系

`bianma-app` 公有仓提供开源核心 / 社区层。
完整官方产品能力可由官方版本提供；公有仓文档不应暴露私有实现细节。

## 协作原则

1. 先保证公有仓文档对外一致、可理解、可执行。
2. 发现文档口径与实现行为不一致时，先记录差异并协调实现核对。
3. 任何超出本边界的内容，应从公有仓主文档中移除或下沉到合适的非公开渠道。
