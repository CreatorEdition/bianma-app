# 1.2 安装指南

## 前置要求

### 必备

- Node.js 18+
- pnpm 8+

### 本地开发或本地构建时

- Rust（用于 Tauri 构建）
- 对应平台的 Tauri 依赖

## 安装 Node.js

### Windows

1. 从 [Node.js 官网](https://nodejs.org/) 下载 LTS 版本。
2. 安装后执行：

```bash
node --version
npm --version
```

### macOS

```bash
brew install node
```

### Linux（Ubuntu / Debian）

```bash
curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -
sudo apt-get install -y nodejs
```

## 拉起项目

在仓库根目录执行：

```bash
pnpm install
pnpm dev
```

如果你需要本地打包：

```bash
pnpm build
```

## 首次验证

1. 应用能正常打开。
2. 可进入“添加供应商”界面。
3. 至少完成一次供应商添加与启用。
