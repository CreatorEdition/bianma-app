# 1.1 软件介绍

## bianma-app 是什么

`bianma-app` 是一个面向 AI 编码 CLI 的桌面控制面板，用于统一管理供应商配置、扩展配置和切换流程。

当前支持的主要 CLI 包括：

- Claude Code
- Codex
- Gemini CLI
- OpenCode
- OpenClaw

## 它解决的核心问题

- 多个 CLI 的配置格式和生效机制不一致，手动维护成本高。
- 在官方登录、API Key、第三方供应商之间切换容易出错。
- MCP、Prompts、Skills 等扩展配置分散在不同路径，缺少统一入口。

## 开源核心定位

`bianma-app` 公有仓聚焦开源核心、社区层、兼容层和公开接入文档。
完整产品层能力、私有发布流程和内部运营细节不在本仓库主文档展开。
